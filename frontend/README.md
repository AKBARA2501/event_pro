# Artify Aura Frontend

Professional event management website with 12 premium pages built with React and modern design.

## Features

✨ **12 Premium Pages**
- **Home** - Landing page with featured events
- **Events List** - Browse and filter events with search functionality
- **Event Details** - Detailed view of specific events with booking
- **Create Event** - Event creation form with step-by-step guide
- **My Events** - Manage your created events
- **Bookings** - View and manage ticket bookings
- **User Profile** - Profile management and account settings
- **Dashboard** - Analytics and event overview
- **Settings** - Notification and preference settings
- **Help Center** - FAQ and support resources
- **Contact** - Contact form and office locations
- **About** - Company information and team

## Tech Stack

- **React 18** - UI library
- **React Router v6** - Client-side routing
- **Vite** - Build tool
- **CSS3** - Modern styling with gradients and animations
- **Responsive Design** - Mobile-first approach

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
frontend/
├── src/
│   ├── pages/          # 12 page components
│   ├── components/     # Reusable components (Navbar, Footer)
│   ├── styles/        # Global CSS
│   ├── App.jsx        # Main app with routing
│   └── main.jsx       # Entry point
├── index.html         # HTML template
├── vite.config.js     # Vite configuration
└── package.json       # Dependencies
```
